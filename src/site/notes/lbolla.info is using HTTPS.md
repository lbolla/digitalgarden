Today I have enabled `https` on this website. I am using certificates issued by [Let\'s Encrypt](https://letsencrypt.org/), which is a new free, automated and open certificate authority.

To manage the certificates, I am using [simp\\~le~](https://github.com/kuba/simp_le/), a simple command line client to issue and renew certificates using [Let\'s Encrypt](https://letsencrypt.org/) API.

All working fine, so far!
