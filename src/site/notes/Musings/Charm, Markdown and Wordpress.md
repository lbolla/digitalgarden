Following my [[Musings/Blog post with Charm\|previous post]], a nice enhancement is using [Markdown](http://daringfireball.net/projects/markdown/) to filter your post before submitting it to [Wordpress](http://wordpress.org/.org).

You can either filter it in your preferred editor ([vim](http://www.vim.org/) uses the `!` command) or invoke the formatter (for example, `markdown`) from [charm](http://ljcharm.sourceforge.net/): command `x`.

Nice!
