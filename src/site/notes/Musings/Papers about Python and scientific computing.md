Here is a list of articles about Python and scientific computing appeared in [Computing in Science and Engineering](http://www.computer.org/portal/site/cise/index.jsp), available online:

-   [Python for Scientific Computing](http://www.computer.org/portal/site/cise/menuitem.92a12adebee18778161489108bcd45f3/index.jsp?&pName=cise_level1_article&TheCat=1015&path=cise/2007/n3&file=oli.xml&) by Travis E. Oliphant ([pdf](http://www.computer.org/portal/cms_docs_cise/cise/2007/n3/10-20.pdf));
-   [Python: Batteries included](http://www.computer.org/portal/site/cise/menuitem.92a12adebee18778161489108bcd45f3/index.jsp?&pName=cise_level1_article&TheCat=1015&path=cise/2007/n3&file=gei.xml&) by Paul F. Dubois;
-   [A Python module for modeling and control design of flexible robots](http://www.siue.edu/~rkrauss/pdfs/python-flexible-robots.pdf) by Ryan W. Krauss;
-   [Analysis of Functional Magnetic Resonance Imaging in Python](https://cirl.berkeley.edu/twiki/pub/User/JarrodMillman/nipy-cise2007.pdf) by K. Jarrod Millman and Matthew Brett, Computing in Science and Engineering, vol. 9, no. 3, pp. 52-55, May/June, 2007;
-   [IPython: A System for Interactive Scientific Computing](http://amath.colorado.edu/faculty/fperez/preprints/ipython-cise-final.pdf) by Fernando Perez;
-   [Matplotlib: a 2D Graphics Environment](http://nitace.bsd.uchicago.edu/misc/c3sci.pdf) by John D. Hunter.

Happy reading!
