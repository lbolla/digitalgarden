While I was researching [[Reading/Books/Ulysses\|Ulysses]], to make sure I really wanted to embark in the adventure of reading it, I stumbled upon [The Joyce Project](https://joyceproject.com/1). Since 2012, Ulysses is open domain, so its text if free to publish. The great added value of The Joyce Project is [its extensive set of notes](https://joyceproject.com/notes/index), written by [the late John Hunt](https://joyceproject.com/notes/lk-kJZUBeQi-aLQfQLNv).

In order to enjoy those notes, I created an eBook out of them. The raw text, notes and various scripts to build the eBook are available at https://github.com/lbolla/ulysses.

You can download the eBook directly from [here](https://github.com/lbolla/ulysses/raw/refs/heads/main/ulysses.epub).
