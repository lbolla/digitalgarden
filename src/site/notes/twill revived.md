After reading [how to do automated testing in python](https://docs.google.com/viewer?url=http://adam.therobots.org/talks/testing.pdf), I\'ve started using [twill](http://twill.idyll.org/), by [C. Titus Brown](mailto:titus@idyll.org)

> twill is a simple language that allows users to browse the Web from a command-line interface. With twill, you can navigate through Web sites that use forms, cookies, and most standard Web features.

Unfortunately, development on twill seems to have stopped 3 years ago.

I\'ve [cloned the project on Github](http://github.com/lbolla/twill), fixed some deprecation warnings and added a cool new feature to inspect headers, as well as html.

Testing is more interesting than I thought.
